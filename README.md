~ Dragon-Dino Game ~

A simple and interactive Dino Runner game built using HTML, CSS, and JavaScript.

The player controls a dinosaur that jumps over obstacles to score points and survive as long as possible.

~ Demo ~

http://127.0.0.1:5500/

~ Features ~

- Smooth jump animation

- Real-time score tracking

- Sound effects

- Increasing difficulty (speed increases over time)

- Collision detection (game over system)

~ Technologies Used ~

- HTML

- CSS

- JavaScript
